Captura

Native macOS screenshot app with annotations, OCR, screen recording and Liquid Glass UI.

⬇ Download DMG View on GitHub

Features

✂️

Region & Fullscreen Capture

Hotkey-driven capture. No permissions required. Includes or excludes desktop wallpaper.

🪟

Window Capture

Click any window to capture it with a beautiful highlight overlay.

📜

Scrolling Capture

Auto-scroll and stitch long pages into one tall screenshot.

🎥

Screen Recording

Record as MP4 via ScreenCaptureKit. No quality loss.

✏️

Annotation Editor

Arrows, rectangles, text, highlight, blur, pen. Full undo/redo.

🔍

OCR

Extract text from any screenshot instantly via the Vision framework.

📌

Pin to Screen

Float any screenshot on top of all other windows.

🪟

Liquid Glass UI

Native macOS 26 design language throughout the entire app.

2
