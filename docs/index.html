Installation

Build from source

Requires macOS 13+ and Xcode 15.

# Clone the repo
git clone https://github.com/alxgb5/captura.git
cd captura

# Build
xcodebuild -project Captura.xcodeproj -scheme Captura build

# Run
open build/Release/Captura.app
